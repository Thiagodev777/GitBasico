Diferenças entre Arquivos

SYSTEM DEVELOPED BY ANTONIO THIAGO

git diff Mostra os Arquivos modificados por dentro

git diff --name-only Mostra apenas os nomes dos Arquivos modificados

Como saber apenas oque foi modificado dentro de um determinado Arquivo?

git diff index.html (nome do arquivo)

Como voltar um Arquivo?

git checkout --HEAD --index.html

volta apenas a modificação fazendo com que não a nada a ser salvo
