Oque e o branch?

Branch nada mais e doque a versao do nosso projeto/pasta! Podemos criar diversos branchs por exemplo: Uma pasta do projeto para produtos outra para pagamentos.. E como se fosse uma copia do projeto original mas e feito apenas para testes e alteracões como o trabalho em uma nova versão por exemplo!

git branch Verifica em que branch voce esta!

Para criarmos um digite:

git branch Nome do branch a ser criado

Para entrarmos em um basta digitar:

git checkout Nome do branch que voce quer entrar
