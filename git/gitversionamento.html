git status

Faz uma varredura na pasta e identifica tudo que foi alterado ou adicionado!

Formas de adicionar um arquivo no GIT:

git add (nome do arquivo a ser adicionado)

git add -A (Adiciona todos os arquivos não salvos)

Commits no GIT!

Commit nada mais e doque pegar todos os arquivos nao salvos e salvar dentro de uma determinada "Versão" para separar varias alterações feitas na pasta!

git commit -m "Descrição do seu commit"

git commit -am "Nome" Adicona o arquivo e da commit ao mesmo tempo!

git log Mostra todos os commits dados na pasta incluindo nome email, data e hora de quem adicionou
