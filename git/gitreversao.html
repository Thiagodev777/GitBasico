Reversões

Para Reversões sempre usamos o Codigo do ultimo commit!

git reset --soft Volta para o ultimo estado soque com os arquivos ja adicionados porem não commitados! (Usado para alteracões)

git reset --mixed Mesma coisa do soft, porem reseta sem os arquivos adicionados e sem o commit

git reset --hard Esqueçe tudo que foi alterado e volta a versão do ultimo Commit
